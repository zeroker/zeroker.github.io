Markdown

1.标题设置：

#字符用于控制标题大小 （一级也就是一个#最大）
2.块注释：（blockquote）

通过在文字开头添加“>”表示块注释

3.斜体： 将需要设置为斜体的文字两端使用1个星号 或者 ““夹起来 @zeroker
4.粗体：
将需要设置为斜体的文字两端使用2个”*”或者”“夹起来 @zeroker
5.链接：
内联方式：This is an [注释] (url).
6.图片：
! [alt text] [id]
